# Pokedex Parcial Web

## 🎯 Objetivo
Proyecto de la primera evaluación parcial. Construye una Pokédex que consulta la **PokeAPI** y muestra **lista**, **detalle** y **favoritos** con **persistencia en el navegador**.

## 🛠️ Tecnologías utilizadas
- HTML
- CSS (metodología BEM)
- JavaScript (ES6+)
- [PokeAPI](https://pokeapi.co/)

## 🚀 Cómo ejecutar el proyecto
1. Clona este repositorio o descárgalo.
2. Abre `index.html`:
   - Opción 1: doble clic (se abre en el navegador).
   - Opción 2 (recomendada): **Live Server** en VS Code.

## ✅ Funcionalidades
- **Cargar 20** Pokémon iniciales (botón en la barra).
- Búsqueda por **nombre** o **ID**.
- **Detalle** con sprite oficial, tipos, altura, peso, base XP y **estadísticas base**.
- **Favoritos** con `localStorage` (contador + lista lateral).
- Filtros por **tipo**, **switch** de “Solo favoritos”.
- Diseño **responsivo** y semántica **BEM**.
